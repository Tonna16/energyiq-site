Technical Stack

  • React + TypeScript frontend with Tailwind CSS
  • Spring Boot + JPA backend with PostgreSQL/H2
  • Real-time WebSocket connections
  • JWT authentication & guest mode
  • Holt Linear & Holt-Winters forecasting models

AI Forecasting Engine

Data-driven model selection automatically chooses between Holt Linear (trend-based) for new appliances (<14 days) and Holt-Winters additive with 7-day seasonality for established patterns. Predictions include smart bounds to prevent unrealistic spikes.

Quick Setup

Clone and run with minimal configuration. Uses H2 database for instant demo mode.

git clone https://github.com/tonna16/TrackEnergy
cd TrackEnergy/backend
mvn spring-boot:run

# In separate terminal
cd TrackEnergy/frontend  
npm install && npm run dev
